# LAB Open Source Builders Fund

The LAB Open Source Builders Fund supports developers building open-source infrastructure across the Bitcoin ecosystem, with a primary focus on empowering African builders.

Created by Let Africa Build (LAB), the fund enables emerging developers to experiment, prototype, and launch tools that strengthen developer infrastructure, payment systems, and decentralized financial rails across Africa.

By supporting early-stage technical experimentation, the fund helps transform promising developers into open-source contributors, infrastructure builders, and startup founders contributing to the global Bitcoin ecosystem.

## Fund Focus

We are particularly interested in projects building:

- Bitcoin developer tooling
- Lightning infrastructure
- Wallet infrastructure and integrations
- Payment rails and financial infrastructure
- Bitcoin Layer 2 experimentation, including Stacks
- Open-source financial tools

## Eligibility

The LAB Open Source Builders Fund prioritizes African developers and teams building open-source infrastructure and tools across the Bitcoin ecosystem.

International builders may also participate if their work collaborates with the LAB ecosystem or contributes meaningfully to the growth of Bitcoin infrastructure across Africa.

Projects should:

- Be open source with publicly accessible code repositories
- Demonstrate a working prototype or clear technical progress
- Build on or integrate with Bitcoin infrastructure
- Include at least one African core contributor or active collaboration with the LAB ecosystem in Africa
- Contribute to infrastructure, developer tooling, financial inclusion, or decentralized systems
- Publish development updates and documentation publicly

The fund is powered by Artizen and begins with an initial $20,000 matching pool to support early builders emerging from the LAB ecosystem.

## How To Apply
Builders interested in the LAB Open Source Builders Fund can apply through the official LAB builder [application form](https://letafricabuild.com/builders-fund).

Developers who prefer to submit via GitHub may open a project proposal using [this template](https://github.com/Let-Africa-Build-LAB/.github/blob/main/ISSUE_TEMPLATE/builder-project-proposal.md).

Please note that Selected projects will be asked to submit their project on the LAB Artizen Fund Page to participate in the matching pool.
